一个轻量级、可部署在 **Railway / Vercel / 本地服务器** 的 PDF 处理 API 服务。  
支持 **PDF 拆分、合并、自定义保存路径、临时文件自动清理** 等能力。

由 FastAPI + PyPDF + APScheduler 构建，已作为可复用 API 工具，可轻松接入前端页面或其他系统。

---

## 🚀 功能 Features

### ✔ PDF 分割（Split PDF）
- 按页范围提取（如 1–1，2–5）
- 自动生成输出文件名
- 支持保存到指定目录（持久文件）
- 支持临时缓存（自动清理）

### ✔ PDF 合并（Merge PDF）
- 多文件合并成一个 PDF
- 支持中文文件名
- 支持保存路径选择

### ✔ 临时文件自动清理（Auto Cleanup）
- **用户未下载 → 3 天后删除**
- **用户已下载 → 24 小时后删除**
- 适合高频 Web 服务，避免服务器文件堆积
